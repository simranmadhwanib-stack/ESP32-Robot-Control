# ESP32 Robot Control 🚗

This project contains ESP32 code for controlling a robot using switch-case logic (Arduino framework).

🏆 Achievement:
Secured 1st Position in Robot Race Competition.

## Features
- F - Move Forward
- B - Move Backward
- L - Turn Left
- R - Turn Right
- S - Stop

## Technology Used
- ESP32
- Arduino Framework
- C Language
